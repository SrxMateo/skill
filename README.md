## 🌟 Visión General
Este repositorio contiene **14 Skills personalizadas** diseñadas para transformar una IA en una agencia tecnológica completa. Cubren todas las fases del ciclo de vida del software: desde la generación de la idea inicial, pasando por el desarrollo Web y Backend, hasta el despliegue automático en servidores de producción (DevSecOps).

## 🛠️ Catálogo de Skills

### 🏗️ Fase 1: Arquitectura y Creación
- **`srxmateo-web-architect`:** Desarrollador Full-Stack. Construye webs con diseño Premium, Local-First y preparadas para +1000 usuarios.
- **`srxmateo-spigot-architect`:** Desarrollador Java Senior. Crea plugins modulares para Minecraft (Spigot/Paper) con bases de datos y soporte asíncrono.
- **`srxmateo-cli-architect`:** Ingeniero de Sistemas. Genera herramientas CLI avanzadas, con banderas, colores y menús interactivos.

### 🔎 Fase 2: Lluvia de Ideas
- **`srxmateo-skill-ideas`:** Estratega Creativo. Analiza el proyecto y te da "10 Ideas Brutales" divididas en: Novedades, Arquitectura, UX/UI, y Monetización.

### 🎨 Fase 3: Auditoría, Mejoras y Adaptación
- **`srxmateo-skill-mejora`:** Inspector, Director de Arte y Copywriter. Audita el código buscando botones muertos y alertas prohibidas, implementa ideas empresariales, mejora textos para SEO y autogenera logos con IA.
- **`srxmateo-skill-optimizador`:** Refactor Ninja. Aplica principios SOLID y Patrones de Diseño al código espagueti.
- **`srxmateo-skill-adaptacion`:** Experto Mobile-First. Transforma páginas de PC en experiencias PWA perfectas para pulgares, con guías de instalación nativa.

### 💻 Fase 4: Datos, Monetización y Marketing
- **`srxmateo-skill-datos`:** Arquitecto de Backend (Big Data). Diseña infraestructuras ultra-escalables con Redis, WebSockets y audita fugas de memoria (Leaks).
- **`srxmateo-skill-monetizacion`:** Arquitecto Financiero. Inyecta pasarelas de pago seguras (Stripe, PayPal, Crypto) asegurando validaciones on-server mediante Webhooks.
- **`srxmateo-skill-publicidad`:** Director de Marketing. Analiza el código buscando novedades y lanza campañas persuasivas (Flyers IA y Copywriting para WhatsApp/Facebook).

### 🚀 Fase 5: Mantenimiento, DevSecOps y Despliegue
- **`srxmateo-skill-repara`:** Francotirador de Código (Debugger). Aplica política de cero desperdicio de tokens: busca líneas específicas y hace cirugía láser en los bugs.
- **`srxmateo-skill-seguridad`:** Hacker Ético (White-Hat). Escanea tu proyecto buscando inyecciones SQL, XSS, y blinda tus rutas contra ataques de fuerza bruta.
- **`srxmateo-skill-vps`:** SysAdmin Avanzado. Gestiona puertos, memoriza cambios históricos, detecta caídas en PM2 y monitorea hardware (RAM, Disco, CPU).
- **`srxmateo-skill-sello`:** DevSecOps Automation. Genera documentación final, commitea a GitHub vía SSH y despliega tu app en el VPS (Nginx + PM2 + SSL).
- **`srxmateo-skill-github`:** Arquitecto de Portafolio. Audita repositorios (públicos/privados), genera e inyecta logos con IA y mejora los READMEs con insignias.

## 🔒 Seguridad
Todas las skills operan bajo la filosofía de seguridad primero (Security First). Nunca debes incrustar contraseñas en texto plano. Las skills te pedirán usar archivos `.env` para manejar tus credenciales privadas.

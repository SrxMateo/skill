## 🛠️ Catálogo de Skills

### 🏗️ Fase 1: Arquitectura y Creación
- **`srxmateo-web-architect`:** Desarrollador Full-Stack. Construye webs con diseño Premium, Local-First y preparadas para +1000 usuarios.
- **`srxmateo-spigot-architect`:** Desarrollador Java Senior. Crea plugins modulares para Minecraft (Spigot/Paper).
- **`srxmateo-cli-architect`:** Ingeniero de Sistemas. Genera herramientas CLI avanzadas.

### 🔎 Fase 2: Lluvia de Ideas
- **`srxmateo-skill-ideas`:** Estratega Creativo. Analiza el proyecto y te da "10 Ideas Brutales" sobre UX/UI y Monetización.

### 🎨 Fase 3: Auditoría, Mejoras y Adaptación
- **`srxmateo-skill-mejora`:** Inspector, Director de Arte y Copywriter. Autogenera logos con IA.
- **`srxmateo-skill-optimizador`:** Refactor Ninja. Aplica principios SOLID y Patrones de Diseño al código espagueti.
- **`srxmateo-skill-adaptacion`:** Experto Mobile-First. Transforma páginas de PC en experiencias PWA.

### 💻 Fase 4: Datos, Monetización y Marketing
- **`srxmateo-skill-datos`:** Arquitecto de Backend (Big Data). Diseña infraestructuras ultra-escalables con Redis y WebSockets.
- **`srxmateo-skill-monetizacion`:** Arquitecto Financiero. Inyecta pasarelas de pago seguras (Stripe, PayPal, Crypto).
- **`srxmateo-skill-publicidad`:** Director de Marketing. Analiza código y lanza campañas persuasivas.

### 🚀 Fase 5: Mantenimiento, DevSecOps y Despliegue
- **`srxmateo-skill-repara`:** Francotirador de Código (Debugger). Aplica política de cero desperdicio de tokens.
- **`srxmateo-skill-seguridad`:** Hacker Ético (White-Hat). Escanea tu proyecto buscando inyecciones SQL y XSS.
- **`srxmateo-skill-vps`:** SysAdmin Avanzado. Gestiona puertos, memoriza cambios históricos y detecta caídas en PM2.
- **`srxmateo-skill-sello`:** DevSecOps Automation. Despliega tu app en el VPS (Nginx + PM2 + SSL).
- **`srxmateo-skill-github`:** Arquitecto de Portafolio. Audita repositorios y mejora los READMEs.
